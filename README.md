# Project SAM — Motion Analysis

This repository contains tools for analyzing and visualizing human arm motion captured using a Vicon motion capture system.

## Features

- Robust Vicon CSV loader
- 3D arm kinematic visualization
- Sagittal plane (YZ) analysis
- Multi-frame snapshot plotting (2×3 layout)
- Joint angle computation (Elbow → Wrist vector)
- Marker coordinate extraction

## Scripts

- `analysis.py` — Core data processing and frame analysis
- `vizualize.py` — Sagittal plane visualization
- `vizualize3d.py` — Interactive 3D visualization
